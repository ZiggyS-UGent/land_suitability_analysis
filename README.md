# land_suitability_analysis

This repository contains the data and code of the interactive tools associated with the manuscript "Land use reallocation in agricultural landscapes leading to better adaptation of land use to local geo-ecological conditions: a case study in Zealand, Denmark". The following resources are provided:

1) Web-based interactive tool for determining fuzzy membership functions

https://anonymous-researcher1010.github.io/land_suitability_analysis/fuzzy_arable_cropland.html

https://anonymous-researcher1010.github.io/land_suitability_analysis/fuzzy_permanent_grassland.html

https://anonymous-researcher1010.github.io/land_suitability_analysis/fuzzy_woody_vegetation.html

2) Web-based interactive tool for performing Analytical Hierarchy Process (AHP)

https://anonymous-researcher1010.github.io/land_suitability_analysis/ahp_arable_cropland.html

https://anonymous-researcher1010.github.io/land_suitability_analysis/ahp_permanent_grassland.html

https://anonymous-researcher1010.github.io/land_suitability_analysis/ahp_woody_vegetation.html
